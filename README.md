# dotfiles

제 설정 파일 들입니다. `Windows` , `WSL2` , `macOS` 을 타겟으로 합니다.

해당 리포는 적용법과 같은 간단한 내용만 다루며 개인 노션에서 파편화된 설정 파일을 `source of truth` 달성하는 것을 기대합니다.

때문에 설정 추가 이유 및 자세한 가이드는 개인 노션에 정리되고 있으며 `바로 사용할 수 있는 상태` 의 컨디션을 기대합니다.
